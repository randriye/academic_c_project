Objective : Create a directory with an hashing table

Each contact has an name, we then compute his hash value with a certain hash function and get the remainder of the number by the division by the length of the hash table. We then insert him in the simply linked list corresponding to this remainder.


Team : Solo

Time Spent : ~ 30 hours

Technologies used : 
- C, valgrind
- Arrays, Custom structures, Pointers

Features : 
- Search, Add, Create, Delete, Modify
- Automatically Resized directory
